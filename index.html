Course Outline

สัปดาห์ หัวข้อ Lecture Workshop / Lab
1 Introduction to Modern Web Development ติดตั้งเครื่องมือพัฒนาเว็บ (VS Code, Node.js, Browser DevTools) และสร้างเว็บเพจแรก
2 HTML Fundamentals สร้างหน้าเว็บไซต์โปรไฟล์ส่วนตัวด้วย HTML
3 CSS Fundamentals ตกแต่งเว็บไซต์ด้วย CSS เช่น color, font, layout
4 Introduction to JavaScript เขียน JavaScript เพื่อโต้ตอบกับผู้ใช้ เช่น ปุ่มกดคำนวณ
5 Advanced JavaScript สร้าง interactive form เช่น calculator หรือ todo list
6 Building Responsive Layouts with Bootstrap and Tailwind สร้างหน้าเว็บไซต์แบบ responsive รองรับมือถือและ desktop
7 📝 Midterm Project Presentations นำเสนอเว็บไซต์ที่พัฒนาด้วย HTML + CSS + JavaScript
8 Introduction to Backend Development using Node.js and Express สร้าง API แรกด้วย Express และเชื่อมต่อกับหน้าเว็บ
9 Version Control with Git สร้าง Git repository, commit, push ไป GitHub
10 Database Integration using SQL I สร้าง database และ table ด้วย SQLite
11 Database Integration using SQL II เขียน query เช่น SELECT, INSERT, UPDATE
12 Database Integration using NoSQL ทดลองใช้ MongoDB และเชื่อมต่อกับ Node.js
13 SQL vs. NoSQL: Advantages and Practical Use Cases เปรียบเทียบการสร้าง database ใน SQL และ NoSQL
14 📝 Capstone Project Presentation I นำเสนอความคืบหน้าโปรเจกต์เว็บไซต์เต็มระบบ
15 📝 Capstone Project Presentation II Demo และนำเสนอผลงาน Final Project
